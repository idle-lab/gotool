一个现代化易上手的Go语言泛型工具库，提供类似Java风格的实用操作方法，同时保持Go语言的高性能特性。✨

## 功能特性

### 当前支持的数据结构 ✅

| 数据结构      | 已实现方法                         | 版本     |
|---------------|-------------------------------|--------|
| **Slice**     | `Contains相关方法`, `IndexOf相关方法` | v0.0.1 |
